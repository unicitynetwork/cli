## Key Findings

### Innovation #1: Hybrid Consensus
First blockchain to successfully integrate PoW for security with BFT for speed.

### Innovation #2: Off-Chain Execution
Assets and computation move off-chain with on-chain security guarantees.

### Innovation #3: Single-Input Transactions
Intentional PoW layer simplification to reduce on-chain burden.
